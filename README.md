# Numerical Methods 2026

This repository contains exercises and notebooks developed for the Numerical Methods course at SISSA. It collects exercises related to ordinary differential equations, hyperbolic ODEs, PDE integration, and physics-informed neural networks.

## Repository structure

- `ode_methods/` — numerical methods for ordinary differential equations (harmonic oscillator and Mercury precession)
- `hyperbolic_odes/` — exercises on hyperbolic systems (wave equation in flat space and spherical simmetry)
- `pde_integration/` — numerical integration techniques for partial differential equations (diffusion and Poisson)
- `pinns/` — introductory experiments with physics-informed neural networks (PINN for the armonic oscillator)
